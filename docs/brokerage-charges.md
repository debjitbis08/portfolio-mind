# Charges

## Equity Brokerage

Rs. 20 or 0.1% per executed order whichever is lower, minimum Rs. 5

## Regulatory & Statutory Charges

### Equity

#### Delivery

**STT (Security Transaction Tax)** 0.1% (Buy, Sell)
**Stamp Duty** 0.015% (Buy)
**Exchange Transaction charge** NSE: 0.00297%, BSE: 0.00375%
**SEBI Turnover charge** 0.0001% (Buy, Sell)
**DP charges (per sell transaction)** Depository Rs.3.5 (male) Rs. 3.25 (female) Groww: Rs.16.5 (Rs.0 for < 100 debit value)
**Investor Protection Fund Trust charge** NSE: 0.0001% (Buy, Sell)

#### Intraday

**STT (Security Transaction Tax)** 0.025% (Sell)
**Stamp Duty** 0.003% (Buy)
**Exchange Transaction charge** NSE: 0.00297%, BSE: 0.00375%
**SEBI Turnover charge** 0.0001% (Buy, Sell)
**DP charges (per sell transaction)** 0
**Investor Protection Fund Trust charge** NSE: 0.0001% (Buy, Sell)

### 18% GST

On Brokerage, DP charges, Exchange Transaction charges, IPFT, SEBI Turnover charges, Delayed payment charges, DDPI charges,
KYS Modification request charges, API charges and Auto Square-Off charges.
